Lliascript

synth

Function, Add new synth

synth(stype, id_, keymode="Poly1", voice_count=8, outbus=["out_0", "outbus", 0])

ARGS: 

      stype       - String, the synth type.
                    Use ls(STYPE) to display list of valid types.
      id_         - int, a unique id number.  If a synth with the same stype and id
                    already exists, replace it.
      keymode     - String, the keymode.
                    Use ls(KEYMODE) to display list of valid key modes.
      voice_count - int, number of voices to allocate.  voice_count is only used
                    if selected keymode pre-allocates a finite number of voices.
      outbus      - Tuple (bus-name, synth-param, bus-offset).
