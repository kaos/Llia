Lliascript

ABUS

Constant

abus

Function, create new audio bus.

abus(name, channels=1)

ARGS: 

      name    - String, If an audio bus with the same name already exists, do nothing.
                All buses, buffers, and synths must have unique names.  An attempt to
                create an audio bus with the name of an existing non audio bus object
                produces a warning.

                Llia automatically creates audio buses for the first several "public"
                buses.  These buses correspond to the first n sound card inputs and
                outputs.  These output buses have names out_0, out_1, out_2, ..., out_n.
                Input buses have names in_0, in_1, in_2, ..., in_n.  Where n is
                typically 8.
      
      channel - int, number of bus channels, default 1.
