Lliascript

qbuses

Function, get/display synth buses ('q' for query)

qbuses(sid=None, silent=None)

ARGS: 

      sid    - String, synth ID, defaults to current synth.
      silent - Boolean, if True do not print results.

RETURNS: dict
The return is a dictionary with the following String keys:

  • audio-inputs
  • audio-outputs
  • control-inputs
  • control-outputs
Each dictionary value is a nested list/tuple of form:
((bus1-name, channel-count),(bus2-name, channel-count), ...)
