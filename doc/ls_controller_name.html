Lliascript

controller_name

Function, get/display/change MIDI controller name.

controller_name(ctrl, new_name=None, silent=False)

ARGS: 

      ctrl     - int, MIDI controller number, 0 <= ctrl <= 127.
      new_name - String or None, If String update bus name.
      silent   - Boolean, If True do not display name.

      controller names must be unique and may not be the same as a bus, buffer,
      synth or channel.

RETURNS: String
