Lliascript

buffer

Function, Create or switch to buffer.

buffer(name, frame=1024, channels=1)

ARGS: 

      name     - String, the buffer name.
      frames   - int, number of frames.  For wavetable frames must be power of 2.
      channels - int, number of channels

      If no buffer named name exists, create a new buffer.
      If a buffer named name already exists, make is the 'current' buffer.
      If a non-buffer object named name exists, produce warning.

RETURNS: Boolean
