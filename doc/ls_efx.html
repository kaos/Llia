efx

Function, Create a new effects synth. or mark an existing one as 'current'

efx(stype, id_, outbus=["out_0", "outbus", 0])

ARGS: 

      stype  - String, class of Synth
      id_    - int, a unique identification.
      outbus - Tuple [bus-name, synth-parameter, offset]
    
      An effects synth is just a special case of a regular "instrument:
      synth.  The primary difference is that effect synths will have
      parameters for audio input, and they are typically restricted to the
      Efx key-mode.

      stype must be a recognized effects synth type. Use the
      command  ls(STYPE) to display a list of valid synth types.

      id_ should be an integer which uniquely identifies this effect.  The
      "synth-id" or "sid" for the effect is stype_n where n is the value of
      id_.   If an effect with the same sid exists, replace it with the new
      one.

      outbus sets the audio bus used for output.
         bus-name - must be an existing audio bus name
      
         synth-parameter - the parameter used for output, typically 'outbus'
      
         offset - an integer offset added to the bus number, useful for
                  multi-channel buses.
