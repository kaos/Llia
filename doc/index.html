Llia

Next( install )     Contents     

Llia is an open collection of virtual synthesizers and audio effects with a comprehensive MIDI implementation.   Synthesizers may be freely layered and processed through complex arrays of effects. Effects may also process external signals.   Additionally there are several "control" synthesizers, these do not directly produce audio but instead control other synthesizers and effects in various ways.

Llia is an OSC client for SuperCollider and provides a user interface and MIDI handler. In response to MIDI events Llia sends request to SuperCollider via OSC messages. SuperCollider in turn acts on these request, usually by creating audio.

Features:

  • MIDI Key split
  • MIDI key layering
  • Tuning tables for alternate scales
  • Comprehensive mapping of MIDI data to synth parameters
  • MIDI Program banks
  • Multiple key-response modes
  • Open collection of synthesizers, effects and control-synths
  • GUI mode allows elements to be 'patched' together graphically
  • Python scripting and interactive REPL

Obtaining:

Llia is available in source form from Github at: plewto/Llia
