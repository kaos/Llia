Lliascript

compose

Function, display Python code representing current application state.

compose creates a Python script which may be used to recreate the current state of Llia. The state includes:

  • audio buses
  • control buses
  • buffers
  • active synths

The state does not include:

  • Internal state of any synth, IE their current program.
  • Program bank data

The compose program is displayed either to the terminal or to a text-editor widget where it may be saved or cut and pasted into a text editor.
