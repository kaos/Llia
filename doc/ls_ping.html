Lliascript

ping

Diagnostic ping to test OSC connection.

ping(sid=None)

There are two types of ping: global and synth.
A global ping test basic OSC connection between the client and server.
A synth ping test connection between client and server side representation of a specific synth object.

ARGS: 

      sid  - String or None
             If None, send a 'global' ping message to the server.
             If String, sid must be an existing synth id, send a synth
             level ping between client and server.

RETURNS: Boolean True if ping message was successful.
