Lliascript

keyrange

Function, set/get/print synth active key range.

keyrange(lower=None, upper=None, sid=None)

ARGS: 

      lower - int or None,  0 <= lower. 
      upper - int or None, lower <= upper <= 127.
      sid   - String or None, The synth ID, defaults to current synth.

RETURNS: Tuple (lower, upper)
