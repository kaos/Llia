Keyswitch

Previous( Group )     Next( Keytable )     Home     Contents     Synth Catalog     

Keyswitch is a feature that interprets MIDI note events as program changes. It was primarily developed to take advantage of drum pads on the mini MIDI controller used during development. The current implementation is very basic and will at some point be replaced with something more robust.
