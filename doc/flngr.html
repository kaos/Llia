Flngr

Previous( xover )     Next( tremolo )     Home     Contents     

Flngr is a straight forward flanger effect.

Delay, mean delay time
Depth, modulation depth
Freq, modulation LFO frequency
External, external signal to modulation depth
Feedback
Low, feedback lowpass filter cutoff
High, feedback highpass filter cutoff
Efx Mix, wet signal mix
External Mix, external signal to wet mix
Amp, main amplitude

Buses

inbus, audio input
outbus, audio output
delaybus delay time control input
mixbus wet mix control input
lfoOutbus internal LFO output

Flngr Parameters

  • delay,    mean delay time (0,1)
  • imodDepth,    LFO modulation depth (0,1)
  • imodFreq,    LFO frequency (0,10)
  • xmodDepth,    External signal modulation depth (0,1)
  • feedback,    delay feedback (-1,+1)
  • feedbackLowpass,    feedback lowpass cutoff (20,20k)
  • feedbackHighpass,    feedback highpass cutoff (20,20k)
  • efxMix,    wet signal mix (0,1)
  • xmixScale,    external signal to wet mix (0,1)
  • amp,    main linear amp (0,2)
