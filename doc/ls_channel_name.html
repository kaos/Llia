Lliascript

channel_name

Function, get/display/change MIDI channel name

channel_name(n, new_name=None, silent=False)

ARGS: 

      n        - int, MIDI channel, 1 <= n <= 16.
      new_name - String or None, If String update bus name.
      silent   - Boolean, If True do not display channel name.

      channel names must be unique and may not be the same as a bus, buffer
      or synth.

RETURNS: String
