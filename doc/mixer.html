4 In 2 Out Mixer

Gain Channel volume slider
Pan
Mod slider, sets amplitude modulation depth from external control signal
Mute Turn channel on/off

The small boxes under the Mute buttons are channel name text fields. These are solely for the user's notes and are not saved in any way.

Buses

inbusA, channel A input
inbusB, channel B input
inbusC, channel C input
inbusD, channel D input
outbus1, main output 1
outbus2, main output 2
xbus, control signal input

Mixer Parameters

  • gainA,    Channel A linear amp (0,2)
  • muteA,    Channel A mute switch (0,1)
  • modA,    Channel A modulation depth (0,1)
  • panA,    Channel A pan position (-1,+1)
  • gainB,    Channel B linear amp (0,2)
  • muteB,    Channel B mute switch (0,1)
  • modB,    Channel B modulation depth (0,1)
  • panB,    Channel B pan position (-1,+1)
  • gainC,    Channel C linear amp (0,2)
  • muteC,    Channel C mute switch (0,1)
  • modC,    Channel C modulation depth (0,1)
  • panC,    Channel C pan position (-1,+1)
  • gainD,    Channel D linear amp (0,2)
  • muteD,    Channel D mute switch (0,1)
  • modD,    Channel D modulation depth (0,1)
  • panD,    Channel D pan position (-1,+1)
  • gain1,    Output 1 linear amp (0,2)
  • gain2,    Output 2 linear amp (0,2)
