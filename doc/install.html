Llia Installation

Quick outline of installation process.
  1. Install SuperCollider if not already installed.
  2. Copy top level Llia folder to wherever you wish, we'll call this the install directory
  3. Copy contents of {install}/sc/Extensions/Classes to SuperCollider extensions directory.
    • Linux:
      • system-wide     /usr/local/share/SuperCollider/Extensions/
      • per-user     ~/.local/share/SuperCollider/Extensions/
    • OSX:
      • system-wide     /Library/Application Support/SuperCollider/Extensions/
      • per-user     ~/Library/Application Support/SuperCollider/Extensions/
    • Windows: ? Fixme
  4. In a terminal, navigate to the directory {install}/sc/SynthDefs/
  5. Start the SuperCollider ide with the command:    scide
  6. From the SuperCollider ide load and execute the file {install}/sc/SynthDefs/install_synths.scd

    To manually execute code in SuperCollider, place the cursor just inside the opening parenthesis and press control+Enter keys simultaneously.

  7. Resolve any Python dependencies, form a terminal:
    1. pip install numpy
    2. pip install pillow
    3. pip install mido

    The mido package requires a MIDI backend and there are two possibilities: rtmidi (preferred) and portmidi. These are very common and pre-installed on many systems. If neither is installed consult the rtmidi and portmidi web sites. In the future pygame may be a third option.

  8. Copy Llia configuration file {install}/resources/config to the appropriate location:
    1. Linux: ~/.config/Llia/
    2. OSX: ~/.config/Llia/
    3. Windows: ? Fixme
    Create missing directories if needed.
  9. Edit configuration file for MIDI backend.
    Locate the [MIDI] section of the config file and change the backend value to either
    • backend : mido.backends.rtmidi
    • backend : mido.backends.portmidi
    Whichever library is installed on your system.
