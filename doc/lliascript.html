Lliascript

Previous( compose )     Next( synth_catalog )     Home     Contents     

Llia has a Python scripting feature. With scripts synths, buses and the connections between them may be programatically created or destroyed.

WARNING: Using lliascript it is possible to execute arbitrary Python code, including code which may be malicious. Only use code from a trusted source.

The compose feature was a major motivation for developing lliascript.
