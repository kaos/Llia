Lliascript

pulsetab

Function, creater buffer filled with pulse waveform (special case of wavetab)

pulsetab(name, harmonics=512, skip=2, frames=1024)

ARGS: 

      name      - String, buffer's name.
                 If a buffer with the same name already exists, do nothing.
                 If name is the name of a non-buffer object, produce warning.
      harmonics - int, number of harmonics.
      skip      - int, number of harmonics to skip.
      frames    - int, number of buffer frames, mujst be a power of 2.
