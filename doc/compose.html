Compose

Previous( Config )     Next( lliascript )     Home     Contents     Synth Catalog     

The compose feature allows the current state of Llia to be saved and later recalled. Compose preserves all active synths, buses and the interconnection between them, bank filenames and current program number for each synth, the layout of graph icons. Compose does not preserve the bank lock feature. Graphs recreated form compose data may not have the exact layout as the original, but they should be close.

The compose feature is accessed from the lliascript option of the main window File menu.

This pops up a lliascript text editor.

Clicking the "Compose" button generates Python code which may be used to reconstruct the current state. The code is also printed to the terminal used to launch Llia.

Use the save button or copy and paste into an editor to save a permanent copy.

To execute a compose script, load it into the lliascript editor and click Exec. Alternatively a compose script may be executed by the "batch" command at the Lliascript prompt.
Compose should only be used immediately after starting Llia and before any synths or buses have been created.
