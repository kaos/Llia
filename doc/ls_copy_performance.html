Lliascript

copy_performance

Function, copy synth performance to clipboard.

copy_performance(slot=None, sid=None)

ARGS: 

      slot - int or None, The program number to copy, 0 <= slot < = 127.
             If None, copy the current performance.
      sid  - String or None, the synth id, Defaults to current synth.

Each program contains 'performance' data which may be copied/pasted separately to the clipboard. This data includes:

  • transpose
  • key range
  • bend range and parameter
  • velocity maps
  • aftertouch maps
  • pitchwheel maps
  • keynumber maps
  • controller maps
