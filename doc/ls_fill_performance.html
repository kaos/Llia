fill_performance

Function, Copy performance to range of bank slots.

fill_performance(start, end, sid=None)

ARGS: 

      start - int, starting slot, 0 <= start <= 127.
      end   - int, ending slot, 0 <= end <= 127.
      sid   - String or None, the synth to copy from, defaults to current synth.
