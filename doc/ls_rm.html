Lliascript

rm

Function, remove (undefine) a Llia entity.

rm(name, param="ALL", sid=None)

ARGS: 

      name  - String, object name
      param - String synth parameter, (only used if removing a parameter map)
      sid   - The synth ID, defaults to current synth.

Name may reference any one of:

  • Audio bus
    Automatic audio buses (out_0, out_1, out_2, ..., in_0, in_1, in_2, ...) may not be removed.
  • Control bus
  • Buffer
  • Synth or Efx

Or may be one of these control sources.
  • "aftertouch"
  • "keynumber"
  • "pitchwheel"
  • "velocity"
  • A named MIDI controller
  • An integer MIDI controller number

The default param="ALL" removes all parameter maps for the given control.
If param is any other value, only that specific parameter map is removed.
