Lliascript

bend

Function, set/get synth bend parameter and range. display bend parameters.

bend(range_=None, param=None, sid=None)

ARGS: 

      range_ - Int or None, if int update current bend range in cents.
      param  - String or None, if string change synth parameter used for pitch bend
      sid    - String or None, The synth ID, defaults to current synth

RETURNS: Tuple (range_, param)
