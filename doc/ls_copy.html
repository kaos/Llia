Lliascript

copy

Function, copy synth program to clipboard.

copy(slot=None, sid=None)

ARGS: 

      slot - int or None, The program number to copy, 0 <= slot < = 127.
             If None, copy the current program.
      sid  - String or None, the synth id, Defaults to current synth.
