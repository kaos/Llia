Creating an Instrument

  1. From the main window make sure the "synths" tab is selected.
  1. Select an available instruments.
  2. An "add-instrument" dialog appears. Select the "Poly1" kemode and click OK.

The dialog is replaced with a new group window with a single active, playable, synth.

To see an editor for the synth click one of the tabs to the right of the "Map2" tab.
