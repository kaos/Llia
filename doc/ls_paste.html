Lliascript

paste

Function, paste clipboard contents to program bank.
The clipboard must contain a program of the appropriate type.

paste(slot=None, sid=None)

ARGS: 

      slot - int or None.
             If an int, 0 <= slot <= 127, copy clipboard to that
             specific slot, otherwise use the current slot.
      sid  - String or None, The synth ID, defaults to the current synth.
