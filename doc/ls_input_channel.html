Lliascript

input_channel

Function, set/get/print synth MIDI input channel.

input_channel(chan=None, sid=None)

ARGS: 

      chan - String or int or None, 
             If int  1 <= chan <= 16, change MIDI channel.
             If String, chan must be name of defined channel
             If None, do not alter current value.
      sid  - String or None, The synth ID, defaults to current synth.

RETURNS: int
