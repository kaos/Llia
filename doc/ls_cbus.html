Lliascript

CBUS

Constant

cbus

Function, create new control bus.

cbus(name, channels=1)

ARGS: 

      name    - String, If an control bus with the same name already exists, do nothing.
                All buses, buffers, and synths must have unique names.  An attempt to
                create an control bus with the name of an existing non control bus object
                produces a warning.
      channel - int, number of bus channels, default 1.
