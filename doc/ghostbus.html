Ghostbus is a control signal processor.

Source Signal

The Source Signal section modifies applies to the input signal.

Gate button, If on the source signal is modulated by the signal appearing at the xbus input.
Highpass, turn highpass filter on.
Lag, Set lag processors depth, may be used to smooth out signal.
Scale, set signal scale factor.
Bias, add fixed value to signal.

Delayed signal

The delayed signal section imparts a delay to the source signal. The delay input is taken immediately after the external modulation and before the highpass filter. The only control in the Source Signal area which has effect on the delayed signal is the "Gate" button.

Delay tumbler, sets delay time up to 4.0 seconds.
Highpass button, enables highpass filter on delay output.
Feedback, sets delay feedback.
