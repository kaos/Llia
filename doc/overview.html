Llia Overview

Llia is an OSC client for SuperCollider available here.

Features:

  • An open set of SuperCollider instruments and effects.
  • Comprehensive MIDI implantation.
  • Program banks.
  • Tuning tables for alternate scales.
  • Python scripting.
  • Optional 'headless' mode; may be run with or without a GUI system.
