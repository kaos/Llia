Keytables

Previous( Keyswitch )     Next( Config )     Home     Contents     Synth Catalog     

Keytables map MIDI key numbers to note frequency. The keytable is individually settable for each synth on the Performance tab of the synth editor, though they are not actually part of the Performance.

All Keytables are global and the current implementation does not provide any way to create or modify them. Instead a few preset tables are provided, all of which are tuned to A440. Many of these scales are described here. It is intended that at some point user editable tables will be introduced.

The tables whose names begin with "EQ" are equal tempered scales with various number of notes per octave. The table EQ12 is the standard 12 note Western scale.
