Lliascript

wavetab

Function, create buffer containing wide variety of waveforms.

wavetab(name, harmonics=512, decay=0.5, skip=-1, cutoff=-1, mode="", depth=0.5, frames=1024)

ARGS: 

      name      - String, buffer name,
                  If buffer with same name already exists,do nothing.
                  If a non-buffer object is already named name, produce warning.
      harmonics - int, set number of harmonics.
      decay     - float, harmonic amplitude falloff.
                  For harmonic n, amp(n) = 1/(n^decay)
                  decay = 0 --> amp(n) = 1, for all n.
                  decay = 1 --> amp(n) = 1/(n+1)
                  decay = 2 --> amp(n) = 1/(n+1)^2
      skip      - int, sets which harmonics are skipped.
                  skip = 2 --> (1, 3, 5, 7, 9, ...)
                  skip = 3 --> (1,2,  4,5,  7,8, ...)
                  skip = 4 --> (1,2,3,  5,6,7,  9,10,11, ...)
                  Disable if skip < 0 --> (1,2,3,4,5,6, ...)
      mode      - String, sets quasi filter mode. Possible values are:
                  "lowpass", "highpass", "bandpass" or "bandreject"
                  Any other value disables filter.
      cutoff    - int, filter cutoff in terms of harmonic number.
      depth     - float, filter depth, 0 <= depth <= 1 
      frames    - int, number of buffer frames, must be power of 2.
